# Iron Lady AI Career Guide

This is a simple AI-powered chatbot web application built using Flask.
It helps working women with career guidance, leadership programs, and mentorship information.

## Features
- Clean website-style UI
- AI chatbot interaction
- Programs and contact information
- Responsive design

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Bootstrap

## How to Run
1. Install dependencies
   pip install -r requirements.txt
2. Run the app
   python app.py
3. Open browser
   http://127.0.0.1:5000
